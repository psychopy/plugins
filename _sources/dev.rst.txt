PsychoPy Plugins
=====================================

For developers
---------------

Info here for how to create a plugin


Example plugin project
-----------------------

We strongly encourage prospective plugin developers to fork the psychopy-demo-plugin package as a starting point for their new plugin. That package includes documentation that will automatically build using sphinx and GitHub Actions


Letting people know about your project
---------------------------------------

You can tell us about your work so we can add it in to our list of curated packages in PsychoPy
