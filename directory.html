    PsychoPy Plugins Directory¶

    The following plugins are currently available:

    • Cambridge Research Systems devices [psychopy-crs]: Support for various hardware devices by Cambridge Research Systems Ltd.

    • Cedrus Support [psychopy-cedrus]: Support for various hardware devices by the Cedrus Corporation.

    • PsychoPy VPixx [git+https://github.com/wichmann-lab/psychopy-pixx.git]: PsychoPy extensions for using VPixx devices

    • Black Box Toolkit Support [psychopy-bbtk]: Extension package for adding support for devices by Black Box ToolKit Ltd. to PsychoPy.

    • LabHackers Research Equipment Support [psychopy-labhackers]: Extension package for adding support for devices by LabHackers Research Equipment to PsychoPy.

    • ioLabs Systems Button Box Support [psychopy-iolabs]: Extension package for adding support for ioLabs Systems button boxes to PsychoPy.

    • Brain Products GmbH Device Support [psychopy-brainproducts]: Extension package for PsychoPy which adds support for Brain Products GmbH devices.

    • Gamma Scientific Inc. Support [psychopy-gammasci]: Extension package for PsychoPy which adds support for Gamma Scientific Inc. devices.

    • Photo Research Inc. Support [psychopy-photoresearch]: Extension package for PsychoPy which adds support for Photo Research Inc. devices.

    • LabJack Support [psychopy-labjack]: Extension package for adding support for LabJack devices to PsychoPy.

    • Konica Minolta Support [psychopy-minolta]: Extension package for adding support for Konica Minolta light-measuring devices to PsychoPy.

    • Emotiv EEG [psychopy-emotiv]: Support for various hardware devices by Emotiv Inc.
